Multi-Class Brain Tumor Classification using Custom Convolution Neural Networks (CNN)
This project classifies brain tumors using a custom CNN architecture.
## Objectives To detect tumors without pre-trained models.
## Tech Stack.
* Python
* PyTorch.
## Results
Final F1 Score:0.7971
## Methodology
The model uses 3 Convolutional blocks and 2 Dense layers.
